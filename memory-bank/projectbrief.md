# Project Brief (项目摘要)

## 核心目标

**SimRun** 是一款专注于跑步记录的跨平台移动应用，采用 React Native + Expo 技术栈。核心理念是「简单跑步，安全记录，专注每一步」——没有社交干扰，没有广告打扰，只有纯粹的跑步数据记录与分析。

## 核心功能

### 1. 跑步记录

- GPS 实时轨迹记录与高精度滤波（工业级卡尔曼滤波）
- 后台持续追踪（锁屏也能记录）
- 距离、时长、配速、卡路里实时统计
- 步频监测（通过计步器传感器）

### 2. 数据管理

- 本地 SQLite 数据库存储（完全离线，保护隐私）
- 跑步历史记录管理（查看、删除）
- 数据导入/导出（JSON、GPX 格式）
- iCloud 备份与恢复（可选）

### 3. 数据可视化

- 渐变色路线回放（绿色→黄色→红色）
- 生涯统计数据图表
- 今日/本周/本月跑步统计
- iOS 灵动岛实时数据展示

### 4. 个人设置

- 多语言支持（简体中文/英文）
- 主题切换（浅色/深色/跟随系统）
- 个人资料编辑（昵称、头像、身体数据）
- 语音播报设置（播报频率、内容、声音选择）

### 5. 隐私与安全

- 无需注册登录，零隐私泄露风险
- 数据本地存储，不上传服务器
- 无需网络即可使用核心功能

## 目标用户

- 注重隐私的跑步爱好者
- 不需要社交功能，只想专注记录的用户
- 喜欢简洁、无广告体验的用户

## 技术亮点

- **完全开源**：代码透明可见
- **原生体验**：Expo 构建，接近原生 App 性能
- **现代化架构**：React 19 + Expo 54 + TypeScript
- **工业级滤波**：卡尔曼滤波优化 GPS 精度
